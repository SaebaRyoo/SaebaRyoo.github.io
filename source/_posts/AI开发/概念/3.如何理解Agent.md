# 什么是Agent
目前理解，Agent这个概念最初来自于人工智能领域，人们往往叫它智能代理（Intelligent Agent），所以，许多人把 Agent 翻译成了智能体。

# 基于LLM的Agent
Agent就是一个软件系统。它包含了LLM和一些额外的功能。

它能够通过LLM对任务进行规划，然后通过工具(Tools)来执行外部动作。
